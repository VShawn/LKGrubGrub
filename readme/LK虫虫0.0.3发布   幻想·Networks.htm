LK虫虫0.0.3发布

主要更新内容：

1.添加轻之国度轻小说文库书籍的下载功能

2.修复已知bug

3.取消之前的追加爬取功能，当下载帖子中的小说过长导致翻页时，将无法下载所有的小说内容，此修改是为了减少程序逻辑复杂度。

4.当选择不下载图片时，以前版本中书籍里的图片标志：[shawn]图片1[lk虫虫]将不会显示。


使用说明：

A.帖子中的小说

1.将小说帖子地址（例如http://www.lightnovel.cn/thread-784629-1-1.html）复制到本程序的地址栏


2.点击加载按钮

3.等待加载完成，登录你的ID

4.点击“新爬虫”按钮，选择保存的目录，开始下载小说。

B.轻小说文库中的小说

1.将某卷书籍的主目录地址（例如http://lknovel.lightnovel.cn/main/book/2920.html）复制到本程序的地址栏


2.往后步骤同A




下载地址：http://pan.baidu.com/s/1c02EKec

详细使用教程参考：http://singlex.sinaapp.com/lkgrubgrub0-0-1/
